## Raw Notes

### #1 — Auto BOQ (Ravi, BuildCraft)
"We spend 2-3 days manually counting items from drawings. Need auto BOQ with material pricing. Should update when design changes. Export to Excel."

### #2 — Client Comments on 3D (Sneha, DesignHub)
"Clients can't give feedback on 3D renders. We export as images and use email threads. Need something like Figma comments but for 3D spaces."

### #3 — AR Export (Amit, Livspace)
"Clients want to see furniture in their actual room using phone camera. We lose deals because competitors offer this. Even a basic AR viewer would help."

### #4 — Custom Furniture Library (Priya, Arch & Beyond)
"We design custom furniture for every project. Need to upload our own 3D models (OBJ/FBX) and use them in floor plans. Current library is too generic."

### #5 — Multi-floor Support (Deepak, Space Matrix)
"We do office spaces spanning 3-4 floors. Currently each floor is a separate project. Need to link them — stairwell alignment, elevator shafts, shared utilities."

### #6 — Mood Boards (Kavya, Studio Organica)
"Would be nice to have a mood board panel alongside the design canvas. Pin reference images, color palettes, material swatches. Currently we use Pinterest separately."

### #7 — Structural Calculations (Farhan, Edifice)
"Can the tool calculate load-bearing walls and beam placement? We currently consult a structural engineer separately. Would save time if basic calculations were built in."

### #8 — Bulk Material Swap (Meera, Interior Matters)
"When a client changes their mind on wood finish, I have to update every single piece of furniture individually. Need a 'find and replace' for materials."

### #9 — Real-time Co-editing (Rohan, CasaCraft)
"Two designers need to work on the same project at once. Right now we take turns and merge manually. Google Docs style collaboration for floor plans."

### #10 — Client Approval Workflow (Anita, NestIN)
"Need to send design to client → they review → approve or request changes → track who approved what. Currently using WhatsApp and losing track."
