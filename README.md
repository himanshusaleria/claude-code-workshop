# Claude Code Workshop

> Build your own AI agent in 2 hours — no coding required.

## Before the Workshop (do this now, takes ~10 min)

### Step 1: Sign up for Claude (free for 1 week)

**https://claude.ai/referral/I6jwoLOL1A**

Create your account and log in. You'll use this during the workshop.

### Step 2: Install these 3 things

**1. Xcode Command Line Tools (Mac only — takes a few minutes, do this first):**
```
xcode-select --install
```
Open Terminal (`Cmd + Space` → type "Terminal" → Enter), paste the above, and click "Install" on the popup. This takes a few minutes — start it first, then continue below.

**2. Node.js:**
Go to **https://nodejs.org** → click the green **LTS** button → run the installer.

**3. Claude Code:**
Open Terminal (Mac) or PowerShell (Windows) and paste:
```
npm install -g @anthropic-ai/claude-code
```

### Step 3: Think about your workflows

Think of **2-3 repetitive tasks** from your job. We'll turn these into AI skills during the workshop.

| Role | Examples |
|------|----------|
| Product | Analyzing call transcripts, summarizing feature requests |
| Sales | Validating leads against ICP, writing outreach emails |
| Marketing | Creating ad copy, writing blog drafts |
| Ops / PM | Status updates, meeting summaries |

### Step 4: Arrive 15 minutes early

We'll help troubleshoot any setup issues before we start. If installation didn't work — no stress, just come early.
